### 6. MVP Architecture Using OpenAI Only

#### Components:
- **Frontend**: Simple web app for image upload and ad preview.
- **Backend**: 
    - Receives image.
    - Uses OpenAI Vision API to analyze product (extract features, determine product type, key style points).
    - Uses GPT-4 to generate ad copy (“NEW ARRIVAL”, “SHOP NOW”).
    - Assembles the final ad preview.
- **Output**: Delivers image+text ad ready for download or sharing.

#### Flow:
1. User uploads product photo.
2. Backend invokes OpenAI Vision for image understanding.
3. Calls OpenAI GPT to generate compelling copy based on product.
4. Combines image + copy in a simple design.
5. Sends the ad preview back to the frontend.
